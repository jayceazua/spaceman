# Spaceman
### *Computer Science 1.1*

<img src="https://github.com/jayceazua/spaceman/blob/master/spaceman_cs1.1.png">

### Description
Spaceman is a guessing game. There is a mystery word which the user tries to guess one letter at a time.

### To start game
``` $ python3 spaceman.py ```

### Technologies used:
- Python 3.7.0
